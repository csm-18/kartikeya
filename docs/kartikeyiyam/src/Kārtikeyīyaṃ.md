# Kārtikeyīyaṃ

Kārtikeyīyaṃ is the documentation book for the Kartikeya Programming Language, for its toolchain and for the project repo.

---

- In Sanskrit, **Kārtikeyīyaṃ** means a book related to **Kartikeya**.
